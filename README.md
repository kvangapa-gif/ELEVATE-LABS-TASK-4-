# ELEVATE-LABS-TASK-4-

Flask User API
This project is a simple REST API that lets you manage user information like name, email, and age. It’s built using Flask, a lightweight Python framework. All user data is stored in-memory, making it perfect for learning and quick testing.

What does it do?


You can:

List all users

Get details for a specific user

Add new users

Update user info

Delete users

Available Endpoints
GET /users – Returns a list of all users.

GET /users/<id> – Fetches one user by their ID.

POST /users – Creates a new user.

PUT /users/<id> – Updates an existing user.

DELETE /users/<id> – Deletes a user.


How to run

1. Make sure Flask is installed:
pip install flask

2. Start the app:
python app.py

3.The API will be live at:
http://localhost:5000
